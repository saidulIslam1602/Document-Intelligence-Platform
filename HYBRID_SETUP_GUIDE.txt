========================================
TROUBLESHOOTING
========================================

"OpenAI API call failed":
  - Check API key in .env.local
  - Verify account has credits
  - Check internet connection
  - Services auto-fallback to mocks

"Services won't start":
  - Check ports: lsof -i :3000,8003
  - Restart Docker: docker-compose down && up
  - Check logs: tail -f /tmp/di-*.log

"No real AI responses":
  - Verify USE_MOCK_SERVICES=false
  - Check OPENAI_API_KEY is set
  - Run: python3 scripts/test_openai.py
  - Restart services
